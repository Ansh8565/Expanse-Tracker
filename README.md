# 💰 Expense Tracker Web Application

## 📌 Overview

The Expense Tracker is a web-based application developed using Flask and SQLite that helps users manage their daily expenses efficiently. It allows users to record, update, and analyze their spending in a simple and user-friendly interface.

---

## 🚀 Features

* 🔐 User Authentication (Login & Registration)
* ➕ Add Expenses
* ✏️ Edit and Update Expenses
* 🗑️ Delete Expenses
* 🔍 Search and Filter Expenses
* 📊 Category-wise Expense Summary
* 📈 Data Visualization using Chart.js
* 🌙 Dark Mode Support
* 📱 Responsive and Modern UI
* ✨ Animated Dashboard Interface

---

## 🛠️ Tech Stack

* **Backend:** Flask (Python)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** SQLite
* **Visualization:** Chart.js
* **Animations:** AOS (Animate On Scroll)

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/Ansh8565/expense-tracker.git
```

2. Navigate to the project folder:

```
cd expense-tracker
```

3. Install dependencies:

```
pip install flask
```

4. Run the application:

```
python app.py
```

5. Open in browser:

```
http://127.0.0.1:5000/
```

---

## 📊 How It Works

* Users register and log in to the system
* Expenses are stored in SQLite database
* CRUD operations are performed using SQL queries
* Category-wise totals are calculated using GROUP BY
* Data is visualized using charts for better understanding

---

## 💡 Key Highlights
